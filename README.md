## English Short Description

[A rogue-like deck-building game centered on cutting up playing cards to create powerful new combinations. Two cards are cut vertically or horizontally and then taped back together, turning destruction into creation. By combining different card halves and their unique abilities, players build powerful new cards while fighting enemies in a paper-crafted world. Inspired by classic card games and cartoon aesthetics, the game dares players to break every collector’s rule to survive and win.]

## German Short Description

[Ein Rogue-like-Deckbuilding-Spiel, dessen Kernmechanik das Zerschneiden von Karten ist, um mächtige neue Kombinationen zu erschaffen. Zwei Karten werden dabei vertikal oder horizontal geteilt und anschließend wieder zusammengeklebt: Zerstörung wird so zur Schöpfung. Durch das Kombinieren unterschiedlicher Kartenhälften mit ihren einzigartigen Fähigkeiten entstehen mächtige neue Karten, während sich die Spieler durch eine papiergefertigte Welt kämpfen. Inspiriert von klassischen Kartenspielen und Cartoon-Ästhetik fordert das Spiel die Spieler heraus, jede Sammlerregel zu brechen, um zu überleben und zu gewinnen.]
